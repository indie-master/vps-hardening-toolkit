# MANUAL-INSTALL / РУЧНАЯ УСТАНОВКА

## EN

```bash
git clone https://github.com/indie-master/vps-hardening-toolkit.git
cd vps-hardening-toolkit
sudo chmod +x bin/vps-hardening install.sh uninstall.sh
sudo ln -sf "$(pwd)/bin/vps-hardening" /usr/local/bin/vps-hardening
sudo vps-hardening doctor
```
